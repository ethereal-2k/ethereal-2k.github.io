About Me

I am a systems-focused engineer driven by low-level programming, performance-critical software, and scalable, efficient computing systems. My work sits at the intersection of networking, GPU acceleration, and distributed systems, following data from packets on the wire to memory layouts on hardware and parallel execution on modern accelerators—then translating careful engineering into measurable performance and robustness gains.

I work close to the hardware, reasoning about memory hierarchies, cache behavior, parallel execution models, and the operational realities of large-scale systems. I write and optimize multithreaded C++ and CUDA, build packet-processing pipelines, and benchmark latency/throughput and resource efficiency as first-class constraints.

I have designed secure data exchange for industrial IoT using blockchain-based smart contracts, focusing on fairness, verifiable transactions, and correctness in distributed settings with untrusted parties.

In networking and GPU acceleration, I built source authentication for IP packets on GPUs (RTX 3060/A4000/A100), processing large PCAP datasets, modifying IPv4 headers, and hashing at scale. I tuned global memory coalescing, SM utilization, kernel launch configs, and host–device transfers to reach high throughput.

I implemented and benchmarked radix tries, Patricia tries, DIR-24-8, and DXR-style IP lookup algorithms for lookup/insert/delete under mixed workloads, emphasizing memory efficiency, update performance, and behavior under dynamic, realistic traces.

I also optimize AI inference systems, working with SGLang, vLLM, TensorRT/TensorRT-LLM, and attention kernel optimizations (paged/flash). I profile with Nsight to tune kernel-level behavior, reduce memory pressure, and improve latency/throughput for production-oriented deployments.
